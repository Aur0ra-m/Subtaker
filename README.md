# Subtaker项目介绍



> 用于探测指定域名的子域名接管信息，参考了[subtake](https://github.com/jakejarvis/subtake)。(能说只是为了学Go嘛，狗头)

## 项目特点

- [x] 支持多指纹探测是否可被间接接管
- [x] 支持域名是否可被注册接管探测
- [x] 支持CANME链解析
- [x] 支持MX记录探测
